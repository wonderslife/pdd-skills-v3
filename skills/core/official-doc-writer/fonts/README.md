# 公文字体资源目录

此目录用于存放GB/T 9704-2012《党政机关公文格式》标准所需的字体文件。

## 所需字体清单

| 字体名称 | 用途 | 文件名 | 来源 |
|---------|------|--------|------|
| 方正小标宋_GBK | 用于发文机关标志、标题 | FZXBSJW.TTF, FZXBSJW_GB.TTF | 需单独下载 |
| 仿宋_GB2312 | 用于正文、发文字号 | SIMFANG.TTF | Windows系统自带 |
| 黑体 | 用于一级标题、密级 | SIMHEI.TTF | Windows系统自带 |
| 楷体_GB2312 | 用于二级标题、签发人姓名 | SIMKAI.TTF | Windows系统自带 |
| 宋体 | 用于页码 | SIMSUN.TTF | Windows系统自带 |

## 快速安装

### 方式1：自动安装（推荐）

运行字体安装脚本，会自动从系统复制字体并安装：

```powershell
# Windows系统 - 以管理员身份运行PowerShell
python scripts/install_fonts.py
```

### 方式2：手动安装

#### Windows系统

1. **从系统复制字体**（仿宋、黑体、楷体、宋体）：
   - 打开 `C:\Windows\Fonts`
   - 复制以下文件到此目录：
     - `simfang.ttf` (仿宋)
     - `simhei.ttf` (黑体)
     - `simkai.ttf` (楷体)
     - `simsun.ttc` 或 `simsunb.ttf` (宋体)

2. **下载方正小标宋_GBK**：
   - 方正字库官网：http://www.foundertype.com/
   - 或从其他合法渠道获取
   - 复制字体文件到此目录

3. **安装字体**：
   - 选中所有字体文件 → 右键 → "安装"
   - 或复制到 `C:\Windows\Fonts`

#### macOS系统

1. 双击字体文件
2. 点击"安装字体"

#### Linux系统

```bash
# 复制字体到用户字体目录
mkdir -p ~/.fonts
cp *.ttf ~/.fonts/

# 更新字体缓存
fc-cache -fv
```

## 字体下载资源

### 方正小标宋_GBK

- **官方网站**：http://www.foundertype.com/
- **说明**：方正字库商业字体，需要购买授权
- **替代方案**：某些单位可能有授权，可联系单位信息部门获取

### Windows系统自带字体

Windows系统通常已预装以下字体：
- 仿宋_GB2312 (simfang.ttf)
- 黑体 (simhei.ttf)
- 楷体_GB2312 (simkai.ttf)
- 宋体 (simsun.ttc)

如果缺失，可从其他Windows系统复制或从微软官网下载。

## 验证字体安装

### Windows系统

1. 打开"设置" → "个性化" → "字体"
2. 搜索以下字体确认已安装：
   - 方正小标宋_GBK
   - 仿宋_GB2312
   - 黑体
   - 楷体_GB2312
   - 宋体

### 使用脚本验证

```powershell
python scripts/install_fonts.py
```

脚本会自动检测系统已安装的字体并显示状态。

## 常见问题

### Q: 字体安装后WPS/Word中不显示？

A: 请尝试以下步骤：
1. 重启WPS/Word
2. 重启电脑
3. 确认字体文件完整未损坏
4. 检查字体是否正确安装到系统字体目录

### Q: 方正小标宋_GBK在哪里下载？

A: 方正小标宋_GBK是商业字体，需要：
1. 从方正字库官网购买授权
2. 联系单位信息部门获取（如有授权）
3. 使用单位已购买的字体资源

### Q: Linux/macOS系统字体名称不同？

A: 不同操作系统字体文件名可能不同，脚本会自动检测常见的文件名变体。

## 字体使用规范

根据GB/T 9704-2012标准：

| 公文要素 | 字体 | 字号 |
|---------|------|------|
| 发文机关标志 | 方正小标宋_GBK | 推荐字号 |
| 标题 | 方正小标宋_GBK | 二号 |
| 正文 | 仿宋_GB2312 | 三号 |
| 一级标题 | 黑体 | 三号 |
| 二级标题 | 楷体_GB2312 | 三号 |
| 发文字号 | 仿宋_GB2312 | 三号 |
| 签发人 | 楷体_GB2312 | 三号 |
| 页码 | 宋体 | 四号 |
