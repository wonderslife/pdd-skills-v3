## 一、事故分类体系

### 1.1 按当事人类别分类

```
第一层级：按参与主体
├── Type-A: 机动车之间事故
├── Type-B: 机动车与非机动车事故
├── Type-C: 机动车与行人事故
├── Type-D: 非机动车之间事故
├── Type-E: 非机动车与行人事故
└── Type-F: 单方事故（涉及固定物）
```

### 1.2 按事故形态分类

```
第二层级：按碰撞形态
├── M01: 正面碰撞（迎面相撞）
├── M02: 追尾碰撞（前后相撞）
├── M03: 侧面碰撞（垂直/斜角）
├── M04: 刮擦事故（侧面平行接触）
├── M05: 翻车事故（单方/多方）
├── M06: 撞固定物（护栏/树木/墙壁）
├── M07: 碾压事故（涉及行人/倒地者）
└── M08: 多车连环事故
```

### 1.3 按道路环境分类

```
第三层级：按道路环境
├── E01: 城市普通道路
├── E02: 城市快速路/高架
├── E03: 高速公路
├── E04: 国道/省道
├── E05: 乡村公路
├── E06: 居民区/厂区内部道路
├── E07: 停车场/收费广场
└── E08: 施工路段/临时改道路段
```

## 二、标准化分析流程

### 通用分析SOP（标准作业程序）

```
步骤1: 事故基础信息录入
    ↓
步骤2: 照片/证据材料审查（如有）
    ↓
步骤3: 各方行为逐一合法性审查
    ↓
步骤4: 违法行为与事故因果关联分析
    ↓
步骤5: 过错程度对比与量化
    ↓
步骤6: 责任比例初步确定
    ↓
步骤7: 法律依据匹配与引用
    ↓
步骤8: 输出完整评估报告
    ↓
步骤9: 不确定性复核与免责声明
```

### 步骤详解

#### Step 1: 基础信息录入

使用[第四章信息采集模板]逐项填写，确保无遗漏。

#### Step 2: 照片/证据审查

**照片质量检查清单：**

- [ ] 照片清晰度是否足够（能辨认车牌、痕迹）
- [ ] 是否包含全景照（展示整体环境）
- [ ] 是否包含局部特写（碰撞部位细节）
- [ ] 是否包含路面标线和交通标志
- [ ] 是否包含各方车辆受损情况
- [ ] 拍摄时间是否与事故时间吻合
- [ ] 是否有多个拍摄角度

**关键要素提取清单：**

- [ ] 车辆最终停止位置及角度
- [ ] 碰撞点位置（路面参照物）
- [ ] 主要碰撞痕迹的位置和方向
- [ ] 路面是否有刹车印/拖痕
- [ ] 散落物分布范围和方向
- [ ] 可见的交通标志和标线
- [ ] 道路宽度、车道数、有无隔离
- [ ] 视线遮挡物（树木、建筑、大型车辆）

#### Step 3: 合法性审查

对每一方当事人的每一个行为进行合法性审查：

```markdown
当事人: [名称]
行为列表:
  1. [行为描述] → 合法/违法 → [法条编号]
  2. [行为描述] → 合法/违法 → [法条编号]
  3. ...

审查结论: 共发现 X 个违法行为
```

#### Step 4: 因果关联分析

对每个已识别的违法行为进行因果测试：

```markdown
违法行为: [描述]
But-for测试: 如果没有此行为，事故是否还会发生？
  □ 是（仅为条件因素）→ 因果权重: 低 (0.2-0.5)
  □ 否（直接原因）→ 因果权重: 高 (0.8-1.0)
  □ 不确定 → 标注: [不确定原因]
```

#### Step 5: 过错量化

```markdown
当事人A过错总值: ____ 分
  - 违法行为1: __分 × 因果系数__ = __分
  - 违法行为2: __分 × 因果系数__ = __分
  - ...

当事人B过错总值: ____ 分
  - ...

过错比值 A:B = __ : __
```

#### Step 6-9: 按报告模板输出

参见SKILL.md中的【报告输出模板】。

## 三、分类型详细指南

### Type-A: 机动车之间事故

#### A1. 追尾型（M02）

**分析重点：**
1. 后车是否保持安全距离
2. 前车是否存在异常操作（急刹、变道、倒车）
3. 前车刹车灯是否正常工作
4. 天气和路面条件的影响

**必查法条：**
- 第43条（安全距离）
- 第22条（安全驾驶）
- 第21条（车辆技术要求）

**常见结果分布：**
- 后车全责：70%
- 后车主责+前车次责：20%
- 其他：10%

#### A2. 十字路口碰撞型（M03/M01）

**分析重点：**
1. 各方信号灯状态（颜色、计时）
2. 是否有交通警察现场指挥
3. 路口是否有让行标志/标线
4. 各方行驶方向和意图（直行/左转/右转）
5. 谁先进入路口

**必查法条：**
- 第38条（信号灯通行）
- 第51条（有信号灯路口通行细则）
- 第52条（无信号灯路口让行）
- 第44条（分道行驶）

**决策树：**

```
有信号灯？
├─ 是 → 各自看什么灯？
│   ├─ 一方红灯一方绿灯 → 红灯方责任
│   ├─ 双方都是绿灯 → 检查流向冲突
│   │   ├─ 直行vs左转 → 左转让直行（第51条第7项）
│   │   └─ 左转vs右转（相对方向）→ 右转让左转
│   └─ 双方都是红灯 → 双方都有过错
└─ 否 → 按让行规则
    ├─ 有标志标线 → 按标志标线
    └─ 无标志标线 → 让右原则（第52条第2款）
```

#### A3. 变道刮擦型（M04）

**分析重点：**
1. 变道方是否提前开启转向灯（≥3秒为宜）
2. 变道时两车相对位置和速度差
3. 变道方是否充分观察后视镜
4. 被变道方是否存在故意加速或不变道空间挤压
5. 变道发生的道路区域（实线/虚线/导流线）

**必查法条：**
- 第45条（变更车道）
- 第22条（安全驾驶/文明驾驶）
- 第30条（交通标线遵守）

#### A4. 超车事故型

**分析重点：**
1. 超车地点是否允许超车（实线区/路口/弯道/桥隧等）
2. 超车时机是否安全
3. 被超车是否有阻碍行为
4. 是否存在对向来车风险

**必查法条：**
- 第43条（超车限制情形）

### Type-B: 机动车与非机动车事故

**分析重点前置：**
1. **首先确定非机动车属性**：
   - 是否符合《电动自行车安全技术规范》（GB17761-2018）新国标？
   - 最高时速≤25km/h？
   - 整车质量≤55kg？
   - 是否有脚踏骑行功能？
   - 如超标 → 按**机动车**处理

2. **确定行驶位置**：
   - 是否在非机动车道内？
   - 是否有非机动车道？
   - 如无非机动车道 → 是否靠右行驶？

**必查法条：**
- 第57条（非机动车道行驶）
- 第76条（赔偿责任的特殊规定）
- 第47条（机动车让行义务，混合道路适用）

### Type-C: 机动车与行人事故

**分析重点：**
1. **行人通行位置**：斑马线/非斑马线/路口/路段中间
2. **行人行为状态**：正常通行/闯红灯/翻越护栏/突然冲出
3. **机动车状态**：是否减速/是否鸣笛/是否采取避让措施
4. **特殊人群**：儿童/老人/残疾人（注意义务更高）

**必查法条：**
- 第47条（人行横道让行 — 核心条款！）
- 第61-63条（行人通行规则）
- 第76条（责任减免条款）
- 第62条（行人过街规则）

**特殊考量：**
- 弱势方保护原则：机动车对行人负有更高的注意义务
- 即使行人有过错，机动车通常也需承担一定责任
- 儿童突然冲出：机动车通常承担主要责任
- 完全免责的唯一情形：行人故意碰撞（第76条第2款末句）

### Type-F: 单方事故

**分析重点：**
1. 是否涉及第三方（即使未直接碰撞）
2. 道路设施是否完善（标志/标线/照明/护栏）
3. 车辆自身是否存在机械故障
4. 是否受其他车辆影响（急刹/挤别/抛洒物）

**常见单方事故类型：**
- 撞护栏/隔离墩
- 冲出路面/翻车
- 撞路边树木/电线杆
- 避让动物/散落物失控
