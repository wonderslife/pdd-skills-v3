## 一、责任判定基本原理

### 1.1 判定的两大核心维度

```
责任判定 = 违法行为识别 × 因果关系强度
         + 过错程度比较 → 责任比例确定
```

**维度A：行为违法性**
- 该行为是否违反了道路交通法律法规
- 违反的是哪一条具体条款
- 违法的性质（作为/不作为）

**维度B：因果相关性**
- 该违法行为是否是导致事故的直接原因
- 还是间接原因或条件因素
- 如果没有该违法行为，事故是否仍会发生（"若无则不"测试）

### 1.2 责任认定的官方标准（《道路交通事故处理程序规定》第60条）

| 责任类型 | 定义 | 适用条件 |
|---------|------|---------|
| **全部责任** | 一方过错导致事故 | 仅一方有违法行为且对方无过错 |
| **主要责任** | 过错程度明显较大 | 一方过错 > 另一方过错 |
| **同等责任** | 双方过错相当 | 双方过错程度大致相等 |
| **次要责任** | 过错程度较小 | 一方过错 < 另一方过错 |
| **无责任** | 无任何过错 | 该方完全无违法行为 |

### 1.3 特殊推定规则

以下情况直接推定为**全部责任**：
1. **逃逸**：事故后逃离现场（第61条第1款）
2. **破坏现场**：故意破坏、伪造现场、毁灭证据（第61条第2款）
3. **故意行为**：一方故意造成事故（第60条第3款）

## 二、过错程度量化参考矩阵

### 2.1 单项违法行为过错分值参考

| 违法行为 | 过错分值(1-10) | 典型责任影响 |
|---------|---------------|-------------|
| **严重违法（8-10分）** | | |
| 酒后/醉酒驾驶 | 10 | 极高，通常主责以上 |
| 逃逸 | 10 | 直接推定全责 |
| 无证驾驶+肇事 | 9 | 显著加重 |
| 故意逆行 | 9 | 通常主责以上 |
| 闯红灯 | 8-9 | 通常主责以上 |
| **较重违法（6-7分）** | | |
| 严重超速（50%以上） | 7 | 加重责任 |
| 违规变道引发碰撞 | 7 | 通常主责 |
| 未按规定让行 | 6-7 | 通常主责 |
| 违章掉头 | 6 | 主责或同责 |
| **一般违法（4-5分）** | | |
| 轻微超速（20%以内） | 4 | 视情况而定 |
| 未保持安全距离 | 5 | 追尾时主责 |
| 未打转向灯变道 | 4 | 减轻但不免除 |
| **轻微违法（1-3分）** | | |
| 未系安全带 | 1-2 | 与事故因果弱 |
| 证件轻微瑕疵 | 1 | 通常不影响定责 |
| 车辆技术状况轻微不符 | 2 | 需证明因果关系 |

### 2.2 综合过错计算模型

```
综合过错值 = Σ(单项违法行为分值 × 因果系数) × 情节系数

其中：
- 因果系数：直接原因=1.0，间接原因=0.5-0.8，条件因素=0.2-0.5
- 情节系数：初犯=1.0，累犯=1.2，恶劣情节=1.3-1.5
```

**责任比例换算：**

| 过错比例关系 | 责任划分建议 |
|------------|-------------|
| A:B ≈ 9:1 或更高 | A:全部/主要, B:次要/无 |
| A:B ≈ 7:3 ~ 8:2 | A:主要, B:次要 |
| A:B ≈ 6:4 ~ 4:6 | A:同等, B:同等 |
| A:B ≈ 3:7 ~ 2:8 | A:次要, B:主要 |
| A:B ≈ 1:9 或更低 | A:次要/无, B:全部/主要 |

## 四、因果关系分析方法论

### 4.1 "条件-原因"二分法

```
条件（Condition）：为事故发生提供了可能性的因素
    例：路面湿滑、视线不良、夜间行驶

原因（Cause）：直接导致事故发生的决定性因素
    例：闯红灯、超速、违规变道
```

**判定原则**：只有"原因"级别的违法行为才应作为定责的主要依据，"条件"级因素可适当调整责任比例。

### 4.2 "若无则不"测试法（But-for Test）

针对每一方的每个违法行为，进行如下思考实验：

> **"如果当事人X没有实施违法行为Y，事故还会以同样的方式发生吗？"**

| 测试结果 | 含义 | 处理方式 |
|---------|------|---------|
| 若无则不会发生 | 该行为是事故的**必要原因** | 作为定责的重要依据 |
| 若无仍会发生 | 该行为仅是**附带条件** | 降低其在定责中的权重 |
| 无法确定 | 存在**事实不清** | 应标注不确定性 |

### 4.3 近因 vs 远因分析法

```
近因链（直接导致事故的行为链条）：
   超速 → 反应距离不足 → 制动不及 → 碰撞 ← 近因

远因链（背景性因素）：
   疲劳驾驶 → 注意力下降 → 超速 → ... ← 近因 ← 远因
```

**原则**：远因不阻断近因的责任效力，但远因的存在可适当增加远因行为人的责任份额。

### 4.4 风险创设理论

衡量各方的行为是否不合理地增加了事故风险：

| 行为类型 | 风险创设程度 | 定责考量 |
|---------|-------------|---------|
| 明显违反安全规范 | 高风险 | 重点考量 |
| 处于法律灰色地带 | 中等风险 | 结合具体情况 |
| 符合一般注意义务 | 低风险 | 通常不定责 |
| 超出一般注意义务 | 零/负风险 | 可能减轻责任 |

## 五、典型案例对照库

### 案例1：典型的追尾事故

**案情**：甲车在城市主干道正常行驶（60km/h，限速60），乙车在前方同车道行驶，乙车突然发现前方障碍物急刹车，甲车制动不及追尾。

**分析**：
- 甲车：未保持安全距离（第43条）— 直接原因
- 乙车：急刹行为是否合理？如为避险则为合法行为

**判定参考**：甲车主要责任（70%-90%），乙车无责任或次要责任（如急刹存在不当）。

### 案例2：十字路口红绿灯冲突

**案情**：甲车沿东西向绿灯直行通过路口，乙车沿南北向左转（左转箭头为红灯，但直行为绿灯），两车在路口中心相撞。

**分析**：
- 甲车：按绿灯直行，无违法
- 乙车：左转红灯通行（第38条），且转弯未让直行（第51条第7项）

**判定参考**：乙车全部或主要责任（80%-100%），甲车无责任或次要责任。

### 案例3：变道引发的侧碰

**案情**：甲车在三车道道路的最左侧快车道行驶，欲向右变道至中间车道，开启转向灯2秒后开始变道，此时乙车在中间车道从后方快速接近，两车发生侧面碰撞。

**分析**：
- 甲车：变道行为本身合法（打了转向灯），但需确认安全间距
- 乙车：是否存在超速？是否注意到甲车变道信号？

**判定参考**：复杂案件，可能为同等责任（各50%）或甲方主要责任（60%-70%）。需结合具体速度、变道角度等因素综合判断。

### 案例4：机动车与电动车事故

**案情**：夜间，甲驾驶小轿车在机动车道行驶（开大灯），乙骑电动自行车在机动车道边缘行驶（无灯光、无反光标识），甲车未及时发现乙车而发生碰撞。

**分析**：
- 甲车：未尽到充分观察义务（第22条），夜间行驶需更谨慎
- 乙车：电动自行车走机动车道（第57条），夜间无灯光设施
- 电动车属性：是否符合新国标？如超标则按机动车处理

**判定参考**：如电动车属非机动车——甲车主要责任（60%-70%），乙方次要责任（30%-40%）；如电动车属机动车——可能为同等责任。

### 案例5：行人"鬼探头"

**案情**：甲车在城市道路正常行驶（40km/h，限速50），前方有公交车停靠站台遮挡视线，一名行人突然从公交车前方跑出横穿马路，甲车避让不及撞倒行人。

**分析**：
- 甲车：是否超速？是否保持足够警惕？（第22条）
- 行人：未确认安全即横穿马路（第62条），从未设人行横道处穿越

**判定参考**：此类案件中，通常甲方次要至同等责任（30%-50%），乙方主要至同等责任（50%-70%）。具体比例视甲车当时的具体速度和反应措施而定。
