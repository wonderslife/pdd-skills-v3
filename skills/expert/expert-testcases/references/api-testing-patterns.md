# API测试模式库

## 23种API测试模式

### CRUD模式 (5种)

| 模式 | 测试目的 | 用例数/端点 | 优先级 |
|------|---------|------------|--------|
| P01-列表查询 | 验证分页、筛选、排序 | 4-6 | P0 |
| P02-详情查看 | 验证返回完整数据 | 2-4 | P0 |
| P03-新增创建 | 验证草稿保存+提交+工作流启动 | 3-5 | P0 |
| P04-编辑更新 | 验证编辑+状态联动+流程重启 | 3-4 | P0 |
| P05-删除移除 | 验证状态校验+级联删除 | 2-3 | P0 |

## 模式选择决策树

```
需要测试的API端点?
│
├─ CRUD操作? → P01-P05
│   ├─ 列表? → P01 + P06 + P07
│   ├─ 详情? → P02 + P06
│   ├─ 新增? → P03 + P06
│   ├─ 编辑? → P04 + P06 + P08
│   └─ 删除? → P05 + P06
│
├─ 含工作流? → P10-P14
│   ├─ 提交? → P10
│   ├─ 审批? → P11 + P12 + P14
│   └─ 驳回重提? → P13
│
├─ 含文件上传? → P15-P17
│
├─ 含业务规则? → P18-P20
│
└─ 需数据一致性? → P21-P23
```
